Class and ID Selectors: When and Why to Use Them

The class and id selectors are available for the purpose of allowing you to style elements (such as div) differently, allowing you options. Both of these attributes will be a great way to stylize single or multiple elements.

When and why: class

  • you want to style multiple elements
    • >
  • you need to use a selector more than once on a page

When and why: id

  • you want to style a single, unique element.
